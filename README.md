rfc-reader
==========

rfc-reader

Description
====

This script is used to read RFC's, and search for RFC's by syntax.
Its almost always under development, but is relatively working.

Requirements
====

Less, wget, curl, and a terminal emulator (either Xterm, mrxvt, or urxvt).
